The Web Developer's Cheatsite

HTML

HyperText Markup Language

HTML is a declaritive markup language that utilizes hyperlink embedded text and other forms of content to build and share websites on the World Wide Web. This language acts as the brick and motar to a website's structure. Laying out the foundation for additonal tools to further build upon its current state.

HTML Tags Learned

Tag Name Description
< html > HTML Initial tag to start a HTML document where all other elements are placed
< head > Head Holds the metadata for the webpage
< title > Title Webpage title displayed on browser tab
< link > Link Allows webpage to load other files along with HTML
< body > Body Contains all content to be used on the webpage.
< header > Header Title or main message displayed on webpage
< main > Main Conatiner for the primary content of the body
< div > Division A container used for dividing the webpage into separate sections
< section > Section Semantic divs for containing a classed section of the webpage
< article > Article Used for large areas of text and content
< h1-6 > Headers Used for text such as titles, captions, stand-alone statements, and more
< p > Paragraph Standard tag used for text that isn't too short or long
< a > Anchor Used for embedding content such as text or an image with a hyperlink
< ol > Ordered List Initializes an ordered list to be filled with rank (1,2,3,...)
< ul > Unordered List Initializes an unordered list to be filled with markers
< li > List Item Contains an item for an ordered or unordred list
< img > Image Contains an image
< video > Video Contains a video
< audio > Audio Contains an audio file
< span > Span Like a div but used to be alongside other elements
< table > Table Initialize a container for a table
< thead > Table Head Contains table header elements
< th > Table Header Header used as a table column
< tr > Table Row Initializes a container for a table row
< tbody > Table Body Initializes a container for a table body
< td > Table Data Row-level values for table rows in table body
< footer > Footer Contains info such as copyright nd contact

CSS

Cascading Stylesheet

CSS is a language that gives a web developer the ability to 'style' a webpage. Styling a webpage means giving the structure provided by HTML character. This can be anything from adding a little color to text, to adjusting multiple contents color, shape, size, and positioning. From text and tables to forms of media such as images, videos, and audio.
